# luctus proficiency

This is a weapon proficiency addon aka. "weapon leveling".

By default you need 10.000 XP to gain a weapon level and every kill gets you 100XP (100kills = 1 level)  
By default you also do not get any reward for weapon leveling

The weapon level gets synchronized every 1 second and is displayed on a player's right hand backside.  
There is no cap to levels.
